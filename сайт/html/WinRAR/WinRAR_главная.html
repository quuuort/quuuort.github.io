WINRAR

WinRAR - самая популярная программа для архивирования данных. Она работает с файлами всех популярных форматов. Является универсальным средством для работы на разных компьютерных платформах. В отличии от других программ имеет повышенную степень защиты данных. Совместим со всеми версиями Windows.

Перед установкой программы рекомендуется ознакомиться с истомными требованиями программы WinRAR:


⎯ Поддержка Windows: Windows 95 - 11
⎯ Оперативная память: 3 MB
⎯ Свободное место на диске: 64MB


УСТАНОВКА И НАСТРОЙКА ПРОГРАММЫ

Программу можно скачать на официальном сайте WinRAR, предварительно поменяв язык и выбрав правильную разрядность ОС. После скачивания дистрибутива и открытия файла .exe, появляется окно установщика программы. На этом шаге необходимо указать папку назначения.

⎯ Процессор с частотой от 1 ГГц
⎯ Объем оперативной памяти от 1 Гб
⎯ Свободное место на диске от 1.5 Гб



Далее предлагается настроить программу для дальнейшего использования. Есть возможность создать ярлык на рабочем столе, закрепить в меню «Пуск», создать программную группу и другое.



ИНТЕРФЕЙС

Панель главного меню представлена в виде нескольких вкладок, в каждой из которых есть свой выпадающий список. Функционал вкладок, следующий:

- Файл. Содержит в себе функции, позволяющие открыть архив, выбрать диск, установить пароль по умолчанию, скопировать файлы в буфер обмена.

- Команды. С помощью этой вкладки можно добавлять файлы в архив, извлекать их в указанную папку, смотреть содержимое, переименовывать и удалять данные с информацией.

- Операции. Есть возможность восстановления повреждённых архивов, проверка на наличие вредоносных файлов, просмотр истории и другое.

- Избранное. Позволяет сохранять файлы «в избранное» для быстрого и удобного просмотра часто используемых файлов.

- Настройки. В этой вкладке можно менять интерфейс, очищать историю и журнал работы, импортировать и экспортировать файлы.

- Справка. Содержит в себе информацию о программе, производителе и сайте WinRAR.



- Справка. Содержит в себе информацию о программе, производителе и сайте WinRAR. Панель инструментов позволяет оперативно добавлять, извлекать архивы, проводить тест файлов в архиве, находить файлы в архиве по поиску и восстанавливать повреждённые файлы.



СОЗДАНИЕ

Для создания архива необходимо выбрать файлы для упаковки, нажать правой кнопкой мыши и выпадающем списке выбрать пункт «Добавить в архив»



РЕДАКТИРОВАНИЕ АРХИВА

Для добавления файла в уже существующий архив необходимо открыть программу WinRAR, на рабочем столе программы кликнуть правой кнопкой мыши и выбрать пункт «Добавить файлы в архив». после этого появится окно, где нужно выбрать файлы для добавления.



Для удаления файла из архива нужно выделить файл и нажать на клавишу delete



РАЗАРХИВИРОВАНИЕ

Для извлечения файлов из архива необходимо выделить архив, нажать правой клавишей мыши и выбрать пункт: «Извлечь файлы…». После этого откроется окно для указания пути распаковывания.
