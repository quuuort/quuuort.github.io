WINZIP

WinZip – многофункциональный архиватор для создания и распаковки архивов, сжатия файлов, шифрования данных и других операций с файлами. Работает на компьютерах под управлением Windows и Mac OS. Занимает около 300 МБ места на диске. Есть также версии для смартфонов и планшетов.

WinZip умеет делиться созданными архивами. По электронной почте и в социальных сетях. Что удобно при совместной работе с файлами. Также поддерживает стандартные «архиваторские» функции – выбор степени сжатия, создание многотомных и самораспаковывающихся архивов, шифрование паролем по выбранному алгоритму.

Преимущества и недостатки WinZip:


⎯ WinZip умеет создавать временные архивы с ограниченным сроком доступа к ним
⎯ также умеет сохранять архивы в облако
⎯ автоматически уменьшать ширину и высоту архивируемых изображений, что также уменьшает их размер
⎯ упаковывает только в ZIP, ZIPX и LHA
⎯ сложный перегруженный интерфейс


УСТАНОВКА И НАСТРОЙКА ПРОГРАММЫ

WinZip имеет упрощённую процедуру установки. Программу можно скачать на официальном сайте, выбрав правильную разрядность ОС. После скачивания дистрибутива и открытия файла .exe, появляется окно установщика программы



ИНТЕРФЕЙС



- «Распаковка» содержит информацию о результатах последней команды, о свойствах, диагностике и прочем

- Во вкладке «Zip» расположены настройки с выбором формата файлов WinZip, функция редактирования файла перед добавлением в архив.

- «Управлять» позволяет перемещать, копировать, удалять, стирать, переименовывать файлы, содержащие информацию.

- «Архивировать/Очистить» позволяет настроить периодическое удаление файлов, очищение корзины, папок быстрого доступа и прочее. Можно задать регулярность, точное время и дату очистки.

- «Инструменты» содержит некоторые утилиты, например подключение сканера, камеры, также есть возможность шифрования, конвертирования. Есть функция разделения и объединения, само распаковывания архивов.

-  «Просмотр» позволяет настроить программу для каждого пользователя, можно менять текущий вид, формат, области. Есть два типа интерфейса «по умолчанию», «классический» и многое другое.


РАБОТА С АРХИВАМИ



Для создания архива необходимо выделить файлы, кликнуть правой кнопкой мыши, в выпадающем списке найти программу WinZip, затем выбрать пункт «Создать разделённый ZIP-файл, после этого в указанной папке появится запакованный файл.

Для архивирования с заданными опциями необходимо выбрать пункт «Сжать в..»

Для редактирования архива необходимо найти файл, который хотим добавить, кликнуть правой кнопкой мыши и выбрать пункт «Add to Zip»




Распаковка файла может происходить следующим образом: выделение файлов для распаковки, нажатие правой кнопкой мыши, строка «Распаковка»
