POWERARCHIVER

PowerArchiver – многофункциональный архиватор, включающий удобную навигацию по папкам и архивам, поддержку шифрования и имеющий собственный формат архивов.

Особенности программы:


⎯ Сжатие, шифрование, обмен и резервное копирование ваших данных.
⎯ Поддержка более 60 форматов: ZIP, ZIPX, 7z, RAR, TAR и других.
⎯ Имеет экстремальную степень сжатия.


УСТАНОВКА И НАСТРОЙКА ПРОГРАММЫ

Дистрибутив можно скачать с официального сайта, ознакомившись перед установкой с системными требованиями программы. Есть поддержка Windows / macOS операционных систем. Приложение занимает около 30 МБ на диске.

Для того, чтобы программа появилась на компьютере требуется пройти стандартную процедуру установки.



ИНТЕРФЕЙС И ФУНКЦИОНАЛ

Интерфейс программы представлен в виде трёх стилей.







Панель главного меню представлена в виде нескольких вкладок, каждая вкладка обладает своим набором функций. Например, на вкладке «инструменты» находятся кнопки для объединения архивов, одиночное или пакетное конвертирование архивов, мульти-распаковка, шифрование и дешифрование файлов (рис.4.2.5). Вкладка «Вид» предназначена для настройки программы с учётом вкуса любого пользователя, здесь есть возможность настройки отображения панели инструментов, командной строки, окон просмотра. «Опции» позволяет осуществлять общие настройки программы: очищать историю, менять язык, управлять лицензией и другое.


РАБОТА С АРХИВАМИ



Чтобы создать/распаковать/удалить архив, нужно на рабочем столе программы нажать правой кнопкой мыши, затем на «Новый архив»/ «Извлечь файлы»/ «Удалить".



Для архивирования с заданными опциями необходимо выбрать пункт «Сжать в..»

После этого появится окно с возможностью выбора формата сохранения, метода сжатия, степени сжатия:


⎯ Очень быстро
⎯ Быстро
⎯ Обычный
⎯ Максимум
⎯ Ультра
⎯ Экстремальное
